Atributos de HTML

Los atributos HTML proporcionan información adicional sobre los elementos HTML y siempre se especifican en la etiqueta de inicio. Los atributos generalmente vienen en pares nombre/valor como: nombre="valor".

Categoría Atributo Función Ejemplo
Atributos Globales id="identificador" Especifica un ID único para un elemento <div id="header">
class="nombre-clase" Especifica una o más clases para un elemento <p class="texto-rojo destacado">
style="propiedad:valor;" Especifica un estilo CSS en línea para un elemento <p style="color:blue;">
title="texto" Especifica información adicional sobre un elemento <abbr title="HyperText Markup Language">HTML</abbr>
data-*="valor" Almacena datos personalizados privados para la página o aplicación <div data-user-id="123">
Atributos de Enlaces href="URL" Especifica la URL de la página a la que va el enlace <a href="https://www.ejemplo.com">
target="_blank" Especifica dónde abrir el documento enlazado <a href="url" target="_blank">
rel="nofollow" Especifica la relación entre el documento actual y el enlazado <a href="url" rel="nofollow">
download Especifica que el destino será descargado <a href="archivo.pdf" download>
Atributos de Imágenes src="URL" Especifica la ruta a la imagen <img src="imagen.jpg">
alt="texto" Proporciona un texto alternativo para la imagen <img src="logo.png" alt="Logo de la empresa">
width="valor" Especifica el ancho de la imagen <img src="img.jpg" width="500">
height="valor" Especifica la altura de la imagen <img src="img.jpg" height="300">
Atributos de Formularios type="valor" Especifica el tipo de elemento de entrada <input type="text">
name="nombre" Especifica el nombre de un elemento de formulario <input type="text" name="username">
value="texto" Especifica el valor inicial de un elemento de formulario <input type="text" value="Valor predeterminado">
placeholder="texto" Proporciona una pista sobre lo que se espera en el campo <input type="text" placeholder="Ingrese su nombre">
required Especifica que un campo de entrada debe ser completado <input type="text" required>
