Documentación de HTML, CSS y JavaScript

Esta página está orientada a la documentación de HTML, sus atributos, elementos y TailwindCSS, el framework que se usa para la pagina.

Introduccion al HTML

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).
Cada pagina HTML, consta de un conjunto de etiquetas que representan los componentes básicos de la pagina web. Con ellos creamos una jerarquía que estructura el contenido en secciones, párrafos, encabezados y otros bloques de contenidos. La mayoría de los elementos HTML tienen una apertura y un cierre que utilizan una sintaxis como la siguiente: <tag>contenido</tag>.

Estructura de Página HTML

<html>
<head>
Contiene metadatos e información sobre el documento que no se muestra directamente en la página.
</head>
<body>
Contiene todo lo visible de la página web.
</body>
</html>

Atributos HTML

Los atributos HTML proporcionan información adicional sobre los elementos HTML y siempre se especifican en la etiqueta de inicio.

Lista de atributos HTML comunes y sus funciones
Atributo Descripción Ejemplo
id Especifica un identificador único para un elemento <div id="header">
class Especifica una o más clases para un elemento (referidas en CSS) <p class="important">
style Especifica un estilo CSS en línea para un elemento <p style="color:blue;">
src Especifica la URL de un recurso externo <img src="imagen.jpg">
href Especifica la URL de la página a la que va el enlace <a href="https://ejemplo.com">
alt Proporciona un texto alternativo para imágenes <img src="logo.png" alt="Logo de la empresa">
title Proporciona información adicional mostrada como tooltip <a title="Visita nuestra página">
target Especifica dónde abrir el documento enlazado <a href="url" target="_blank">

TailwindCSS

Tailwind CSS es un framework de CSS de utilidad que permite construir diseños personalizados rápidamente sin salir de tu HTML.

Características principales:

  • Enfoque utility-first para el diseño
  • Altamente personalizable mediante configuración
  • Optimizado para producción con PurgeCSS
  • Diseño responsive integrado
  • Componentes de estado como hover, focus, etc.

Ejemplos de clases comunes:

Espaciado

m-4 Margen de 1rem
p-2 Padding de 0.5rem

Tipografía

text-xl Tamaño de texto extra grande
font-bold Texto en negrita

Colores

text-blue-500 Texto azul
bg-gray-200 Fondo gris claro

Flexbox

flex Display flex
justify-center Centrar horizontalmente

Ejemplo práctico:

<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">
          Botón con Tailwind
        </button>

Responsive Design con Tailwind:

Tailwind incluye prefijos para diferentes tamaños de pantalla:

  • sm: - Pantallas pequeñas (640px y más)
  • md: - Pantallas medianas (768px y más)
  • lg: - Pantallas grandes (1024px y más)
  • xl: - Pantallas extra grandes (1280px y más)
  • 2xl: - Pantallas 2x extra grandes (1536px y más)
<div class="text-sm md:text-base lg:text-lg">
          Este texto cambia de tamaño según el ancho de la pantalla
        </div>
